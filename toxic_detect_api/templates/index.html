Toxic labels

/api/labels/

Description: Lists the currently available toxic label, and provides basic information about each one such as the owner and availability.

Method: GET

Response:

{
    "labels": [
        "toxic",
        "severe_toxic",
        "obscene",
        "threat",
        "insult",
        "identity_hate"
    ]
}

Code Example:

GET curl --location 'https://foxhound-intimate-oriole.ngrok-free.app/api/labels/'

Predict comment

/api/detect

Description: Endpoint to predict toxic comment. The endpoint accepts a list of comments and returns a list of prediction for each comment

Method: POST

Parameters:

  • comments- (required) List comment text

Response:

[

      "comment": "i kill you right now",
      "prediction": {

  },
  {
